# PlantMe - Магазин домашних растений

## Описание проекта
Интернет-магазин домашних растений с функционалом покупки, каталогом и личным кабинетом.

## Стек технологий
- **Backend**: Django + Django REST Framework
- **Frontend**: React.js
- **База данных**: SQLite (разработка), PostgreSQL (продакшн)
- **Верстка**: CSS3, адаптивный дизайн

## Структура проекта
- `/backend` - Django приложение
- `/frontend` - React приложение

Подробные инструкции в папках backend/ и frontend/
